# Simple Interest Calculator

## 📌 Project Description

This project is a simple web-based application that calculates **Simple Interest** based on user input.

It takes:

* Principal Amount (P)
* Rate of Interest (R)
* Time (T)

And calculates the Simple Interest using a standard formula.

---

## 🧮 Formula Used

Simple Interest (SI) is calculated using:

SI = (P × R × T) / 100

---

## 🚀 Features

* Easy-to-use interface
* Instant calculation
* Responsive design
* Beginner-friendly project

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 💻 How to Use

1. Enter the **Principal Amount**
2. Enter the **Rate of Interest**
3. Enter the **Time (in years)**
4. Click on **Calculate**
5. The Simple Interest will be displayed instantly
